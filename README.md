# Dental Crown — Patient Info Sheet

A modern, responsive, single-page patient guide explaining the dental crown procedure, built as a self-contained `index.html` (no external dependencies — just open it in a browser).

## Contents
- **`index.html`** — the webpage. Includes: what a crown is, when you might need one, the 6-step procedure, crown types, aftercare tips, and an FAQ.
- **`dental crown procedure.jpeg`** — the original source infographic.

## Usage
Open `index.html` in any web browser, or host the file on any static web host (GitHub Pages, Netlify, etc.).

## Features
- Fully responsive (mobile → desktop)
- Light & dark mode aware
- Accessible: semantic HTML, keyboard-operable FAQ, respects reduced-motion
- No frameworks, CDNs, or build step — 100% self-contained
